# Delete My App Data

Task Assistant does not require an account. The app uses a pseudonymous installation ID to keep app data separate per installed app.

To delete your app data:

1. Open Task Assistant.
2. Go to Settings.
3. Open the Privacy section.
4. Tap Delete My App Data.
5. Confirm deletion.

This deletes cloud reminders, lists, list items, calendar events, quick reminders, and related app data for the current install. It also clears the local identity/cache and starts the app as a new empty install.

Local display preferences, such as theme, accent color, and clock settings, may remain.
