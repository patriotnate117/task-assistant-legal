## Common Topics

**Reminders or notifications not firing**
Check that notifications are enabled for Task Assistant - with Voice Assist in
your device settings. On Android, also check that Battery Optimization is not
restricting the app in the background.

**Lost my data after reinstalling**
Task Assistant - with Voice Assist uses a pseudonymous installation ID stored on
your device. If you reinstall or clear app data without using the in-app Delete
My App Data option, that ID may be lost and the app will start as a new empty
install. Recovery of data without the original installation ID is not possible.

**How to find my installation ID**
Go to Settings > Copy My Install ID. Include this ID in any support request
so your records can be located.

**Delete my data**
See the [Data Deletion Request](https://patriotnate117.github.io/task-assistant-legal/data-deletion/)
page for in-app and web-based options.

**Ad-related questions**
Task Assistant - with Voice Assist uses Google AdMob for ads. Ad content is
served and managed by Google. For ad preferences, visit
[Google's My Ad Center](https://myadcenter.google.com/). For consent choices
within the app, go to Settings > Privacy > Ad Preferences.
